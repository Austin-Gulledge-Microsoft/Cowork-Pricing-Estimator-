# Cowork Pricing Estimator

This repository contains a directional cost-estimation dashboard for planning and discussion purposes.

## Important

- This is an **estimate**, not a quote or contract.
- Results are **illustrative only** and will vary based on actual usage, model mix, workflow complexity, and pricing changes.
- Assumptions such as **licensed users**, **active usage**, **sessions per user**, and **credit consumption** can materially change the outcome.
- Do not use this output as the sole basis for procurement or budget approval.

## How to use

Open `index.html` in a browser and adjust the controls to compare scenarios.

## Notes

- Values shown are approximate.
- Validate against real usage and current pricing before making decisions.
- Update the assumptions in the dashboard as your deployment evolves.
